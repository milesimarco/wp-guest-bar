Add a customizable guest bar to your WordPress site.

== Description ==

WP Guest Bar displays a customizable WordPress admin bar for non-logged-in users.

Inspired by BuddyPress, it provides quick links (e.g., Log In and Register) for guests, improving user engagement and site navigation.

= Features =

* Home icon with site name and link (mobile-friendly)
* Login link with icon (mobile-friendly)
* Register link (if enabled)
* Customizable message area for guests
* WPML compatible and fully translatable
* Lightweight and performance-optimized

Demo: [wpgov.it](http://wpgov.it)

Perfect for BBPress forums or any site looking to encourage user registration and interaction.

== Installation ==

1. Upload the entire folder to `/wp-content/plugins/` or install via the WordPress plugin screen.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings as needed and enjoy!

== Frequently Asked Questions ==

= Can I customize the guest bar message? =
Yes, you can set a custom message for guests in the plugin settings.

= Is it compatible with my theme? =
WP Guest Bar is designed to work with most themes. For advanced customization, you may use custom CSS.
